# Product Overview

## Purpose
Mumbai Slang Translator - A web application that helps users understand Mumbai-specific slang, street language, and cultural nuances. The app translates local terms into English with cultural context, making it perfect for tourists, new residents, or anyone wanting to understand local Mumbai language.

## Core Features
- **185+ Mumbai Slang Terms** - Comprehensive dictionary from street language to corporate slang
- **Smart Search** - Real-time suggestions and fuzzy matching
- **Cultural Context** - Explanations include cultural meaning, not just literal translations
- **Offline Support** - Progressive Web App with service worker
- **Mobile-First Design** - Mumbai-inspired UI optimized for mobile devices

## Target Users
- Tourists visiting Mumbai
- New residents and migrants
- Students and professionals
- Anyone interested in Mumbai culture and language

## Key Principles
- **Mumbai-Centric**: Always assume Mumbai context, never generic India-wide interpretations
- **Cultural Authenticity**: Provide cultural meaning alongside translations
- **Casual Tone**: Match Mumbai's fast-paced, friendly communication style
- **Practical Usage**: Focus on everyday scenarios and real-world examples

## Slang Categories
- Everyday conversation (bhai, scene, mast)
- Street/Tapori language (bhidu, bindaas, jugaad)
- Local transport (local, fast, virar fast)
- Food culture (cutting chai, tapri, paisa vasool)
- Modern corporate (work ka scene, wfh scene)
- Cultural identity (mumbai spirit, apna time aayega)
